## Week 2 - Bits and Bytes
- Memory = bytes: All data is stored as bits grouped into bytes; interpretation gives it meaning.
- ASCII: ASCII maps numbers to characters (e.g., ‘A’ = 65).

## Week 3 - Structure of Text
- Markup: HTML (tags) and Markdown (symbols like #) structure text.
- Escaping Special characters (<, #, ") must be escaped to show them literally.

## Week 4 - Manipulating Text & Text Editors
- Use text editors such as Visual Studio Code for coding, not Word.
- Text structure matters like Lines, indentation, and whitespace help readability and organization.

## Week 5: CLI
